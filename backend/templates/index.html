Document Processing API

Welcome to the Document Processing API. The following endpoints are available:

Chat API

Endpoint: /api/chat (POST)

Process chat messages and get AI responses.

Document Upload

Endpoint: /api/upload (POST)

Upload and process documents.

Get Documents

Endpoint: /api/documents (GET)

Retrieve a list of uploaded documents.

Process Query

Endpoint: /process_query (POST)

Process specialized queries related to companies or sectors.

Reports

Endpoint: /api/reports/{report_id} (GET)

Get a specific report by ID.

Health Check

Endpoint: /api/health (GET)

Check the API health status.
